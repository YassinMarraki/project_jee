Présentation de Mon Annonce

Cette application web a été faite dans le cadre de notre projet Framework Web (Spring) à l'Ecole Nationale Supérieure d'Informatique et d'Analyse des Systèmes ENSIAS. Ce projet consiste à créer une application web qui va servir à lier les vendeurs et les acheteurs à travers les annonces. Donc chaque personne pourra s'inscrire dans notre plateforme et diffuser son annonce pour que les autres visiteurs du site web puissent la visualiser.
